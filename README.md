# Corona-Dashboard
 Corona News Dashboard (German)

This Dashboard is optimized for a Resolution of 1920 x 1080 and Fullscreen Browsers (F11).

The Videolinks on the left side are the following German Broadcasters:

1. Tagesschau24
2. Welt
3. Deutsche Welle
4. ZDF
